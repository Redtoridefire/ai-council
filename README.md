# AI Council System

Multi-agent AI decision engine using Claude, OpenAI, and Gemini.

## Features
- Parallel multi-agent reasoning
- Red team logic + cyber analysis
- Debate and critique phases
- Chairman synthesis decision

## Run
```bash
python council.py
```

## Environment Variables
- `ANTHROPIC_API_KEY`
- `OPENAI_API_KEY`
- `GEMINI_API_KEY`
